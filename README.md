# 🌈 Paulista Diversa

Plataforma cultural dedicada à memória, visibilidade e ocupação LGBTQIAPN+ na Avenida Paulista, por meio da arte, arquitetura, comunicação e tecnologia.  
Este projeto é um site editorial moderno, responsivo e cheio de identidade visual.

---

## ✨ Funcionalidades

- **Navbar fixa** com links para todas as seções.
- **Hero Section** com destaque para o logo e mensagem principal.
- **Seções temáticas**: Sobre, Mapeamento, Narrativas, Eventos, Galeria e Contato.
- **Cards interativos** com efeitos de hover e gradientes artísticos.
- **Formulário de contato** estilizado e funcional.
- **Design responsivo** para desktop, tablet e mobile.
- **Animações sutis** (logo flutuante, glow pulsante, transições suaves).
